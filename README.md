## [Installation](#table-of-contents)
`git clone` the repo to your local machine. 

Once cloned, run the following command in your terminal:

`npm install`.

## [Usage](#table-of-contents)

Run the following command to seed the files:

`npm run seed`.

Then run:

`npm run develop`
